COHP & COOP Analysis (VASP + LOBSTER + wxDragon)

COHP (Crystal Orbital Hamilton Population) and COOP (Crystal Orbital Overlap Population) are bonding analysis methods used to understand chemical interactions in solids.

This workflow uses:

VASP – Performs DFT electronic structure calculation

LOBSTER – Reconstructs local orbitals and calculates COHP/COOP

wxDragon – Visualizes bond strengths and COHP curves

What COHP/COOP Tell You

Identify bonding vs antibonding states

Quantify bond strength using ICOHP

Analyze orbital contributions

Evaluate structural stability

Interpretation

COHP < 0 → Bonding

COHP > 0 → Antibonding

More negative ICOHP → Stronger bond

COHP is generally preferred over COOP because it has stronger physical meaning.

Applications

Transition metal oxides

Intermetallic compounds

Catalysts

Magnetic materials

Semiconductor bonding analysis
